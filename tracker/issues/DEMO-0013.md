# Problem

The starter could dispatch active issues, but it still left two gaps for trusted-local autonomous backlog flow: file-backed `depends_on` was ignored at runtime, and tracker state changes remained entirely manual.

# Scope

Add dependency-aware admission and optional file-backed state transitions while preserving the existing local-first tracker seam and keeping non-file trackers read-only.
