# Problem

The parser needs a reliable foundation for reading text before feature-specific rules are added. If line-ending normalization, source positions, and block scanning stay implicit, every later feature will carry its own edge-case behavior.

# Scope

Implement source normalization and the block-level parsing foundation needed for later paragraph, heading, list, quote, and code-block work.
