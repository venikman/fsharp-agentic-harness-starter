# Problem

Once inline delimiters exist, the parser needs explicit hyperlink and media forms. Links and reference definitions are semantically rich enough that they should be designed and evidenced as their own bounded step.

# Scope

Implement inline links, images, and the chosen level of reference-definition support for the current dialect boundary.
