# Problem

The parser needs an operator-visible evidence surface before an initial release can be evaluated honestly. A renderer and CLI smoke mode make AST and rendered output inspectable with real sample inputs.

# Scope

Add an HTML renderer for the supported AST subset and a CLI surface that can emit either AST or HTML for real sample files.
