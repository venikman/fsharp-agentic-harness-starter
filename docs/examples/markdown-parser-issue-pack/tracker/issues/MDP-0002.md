# Problem

Later parser work needs a stable internal language. Without a documented AST, source-span model, and public parse result surface, feature issues will bake accidental semantics into code and make review harder.

# Scope

Define the parse tree, source location model, parse-result surface, and the minimal public library entry point. Do not broaden into full renderer behavior or wide dialect coverage here.
